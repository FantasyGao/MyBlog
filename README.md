# 个人博客项目<br>
This is a blog repository ( Node+Mongodb+Express)<br>
 ##安装mongodb数据库（网上有教程）<br>
 ##运行mongodb数据库<br>
 打开浏览器localhost：27017端口验证<br>
 ##下载项目到本地<br>
 git clone  https://github.com/FantasyGao/MyBlog.git<br>
 ##进入目录<br>
 cd MyBlog<br>
 ##运行<br>
 npm start<br>
 ##打开浏览器<br>
 localhost //localhost:80  即可进入页面<br>
